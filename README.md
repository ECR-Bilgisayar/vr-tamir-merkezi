
# VR Kiralama - Professional VR Equipment Rental Platform

A modern, feature-rich VR equipment rental platform built with React, TailwindCSS, and Framer Motion. This application provides a complete solution for VR rental businesses to showcase their products, manage leads, and facilitate bookings.

## üöÄ Features

- **Product Catalog**: Browse VR headsets (Meta Quest 3, HTC Vive XR Elite, PSVR2, Samsung Gear VR) and accessories
- **Game Library**: 100+ VR games with filtering by category, platform, and age rating
- **Rental Packages**: Three tier packages (Starter, Professional, Enterprise) with detailed comparisons
- **Contact Form**: React Hook Form with Zod validation and localStorage persistence
- **Lead Management**: Admin panel to view, filter, export, and manage customer inquiries
- **Responsive Design**: Mobile-first approach with smooth animations
- **SEO Optimized**: React Helmet for meta tags and semantic HTML structure
